# 💫 About Me:
Hi, I'm Samaksha Agarwal, a Computer Science undergraduate at Jaypee Institute of Information Technology, Noida, with a strong interest in backend development and distributed systems.<br><br>I primarily work with Java and Spring Boot, building scalable backend applications, REST APIs, and event-driven systems. My current focus is on understanding how large-scale systems are designed, developed, and maintained, with hands-on experience in technologies such as Apache Kafka, Spring Security, PostgreSQL, MySQL, Hibernate, and Spring Data JPA.<br><br>I enjoy building projects that go beyond basic CRUD applications. Currently, I am developing RainIQ, a distributed multi-service platform that leverages Kafka-based asynchronous communication, JWT authentication, Flyway-managed database migrations, and AI-powered insights. I have also built backend applications including a BookMyShow clone and a Weather Dashboard API service.<br><br>Alongside development, I actively practice Data Structures and Algorithms and have solved more than 500 problems across LeetCode, Codeforces, and CodeChef. My areas of interest include Graph Algorithms, Dynamic Programming, System Design, Microservices, Event-Driven Architecture, and Backend Scalability.<br><br>I believe in understanding systems from the ground up—starting with implementation, exploring the underlying concepts, and then focusing on performance, reliability, and scalability.<br><br>Currently learning: Distributed Systems, Kafka Internals, System Design, Docker, and Production-Grade Backend Architecture.<br>
